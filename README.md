# ARCXP

ARCXP is a contributor engagement platform built for the ARC ecosystem.

It combines wallet identity, XP progression, quests, streaks, referrals, NFT participation, and contributor leaderboards into one ecosystem experience.

## Features

- Wallet Authentication
- Google Login
- XP & Streak System
- Daily GM Check-in
- Referral Rewards
- NFT Minting
- Contributor Leaderboard
- Activity Tracking
- Wallet-based Identity
- Mobile Responsive UI

## Live Demo

https://arcxp.xyz

## Tech Stack

- React
- TypeScript
- Firebase
- WalletConnect
- Web3 Authentication

## Purpose

ARCXP helps increase ecosystem participation through gamified onboarding, quests, rewards, and contributor engagement.

## Status

Public working prototype for ARC ecosystem showcase.
